# Focus Timer with Multi-Task Checklist

## Overview

The **Focus Timer** in **wurk2do** allows you to select multiple tasks from your week and work on them in a distraction-free fullscreen mode with a checklist to track your progress.

## How to Use

### Step 1: Add Tasks to Focus List

From either **Week View** or **Calendar View**:

1. **Click the ‚è± timer icon** on any task card
2. The task is added to your focus list
3. Repeat for all tasks you want to work on
4. The timer bar shows how many tasks you've selected

### Step 2: Enter Focus Mode

1. Click the **Fullscreen button** (expand icon) in the timer bar
2. Alternatively, click on the timer display when you have multiple tasks

### Step 3: Work in Focus Mode

**Fullscreen Display Shows:**

- ‚úÖ **Task Checklist** - All selected tasks with checkboxes
- üéØ **Large Timer** - Prominent circular countdown
- üìä **Progress** - X/Y tasks completed
- ‚≠ê **Priority Indicators** - Visual priority levels
- ‚è± **Time Estimates** - Estimated hours for each task
- üìÖ **Day Labels** - Which day each task belongs to

### Step 4: Complete Tasks

1. **Check off tasks** as you finish them
2. Tasks update in real-time across all views
3. Completed tasks show with strikethrough
4. Progress counter updates automatically

### Step 5: Manage Your Session

**Timer Controls:**
- **Start/Pause** - Control the countdown
- **Reset** - Restart the timer
- **Duration Presets** - 5m, 15m, 25m, 45m, 60m
- **Close** - Exit fullscreen (top-right X)
- **Clear All** - Remove all tasks from focus list

## Tips & Best Practices

### Effective Task Selection

‚úÖ **DO:**
- Group related tasks together
- Mix quick wins with longer tasks
- Add realistic number of tasks (3-7)
- Include tasks you'll actually finish

‚ùå **DON'T:**
- Add tasks you're not ready to work on
- Overload with 20+ tasks
- Mix unrelated work contexts
- Add placeholder/reminder tasks

### Timer Settings

**Short Sessions (5-15 min):**
- Quick tasks
- Between meetings
- Email/admin work
- Code reviews

**Medium Sessions (25-45 min):**
- Focused coding
- Writing/documentation
- Design work
- Planning sessions

**Long Sessions (60+ min):**
- Deep work
- Complex problem-solving
- Large feature development
- Research/learning

### Productivity Hacks

**Pomodoro Technique:**
1. Set 25-minute timer
2. Add 4 tasks to focus list
3. Work until timer ends
4. Take 5-minute break
5. Repeat

**Task Batching:**
1. Add similar tasks (e.g., all emails)
2. Set appropriate timer
3. Complete all at once
4. Save context-switching time

**Priority Focus:**
1. Filter Week View by high priority
2. Add top 3 tasks to timer
3. Complete before checking other work
4. Start day with wins
